jelford -> chest

chest, a python web service for charting arbitrary data series over time.

Installation

git clone https://github.com/jelford/chest.git
pip install chest

Usage

Start the server:

python -m chest.start

Post data:

>>> from requests import post
>>> from json import dumps
>>> post('http://server_address:5805/series_name', data=dumps({'value': 4}))
>>> post('http://server_address:5805/series_name', data=dumps({'value': 5}))
>>> post('http://server_address:5805/series_name', data=dumps({'value': 6}))

Browse to http://server_address:5805/series_name/chart?plot_y=value to see the posted data in a chart, with value on the y-axis
